# Changelog

All notable changes to the "VS Code Instance Rename" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-14

### Added
- Extension icon for Marketplace display
- Comprehensive "How to Use" documentation with step-by-step screenshots
- Visual demo showing command palette usage
- Visual demo showing custom name result

## [0.0.1] - 2026-01-14

### Added
- Initial release
- Command "Rename VS Code Instance" to set custom window titles
- Workspace validation: shows warning if no folder/workspace is open
- Saves window title to workspace settings (.vscode/settings.json)
- Displays current title as default value in input prompt

### Features
- Simple command palette invocation
- Persistent titles across VS Code sessions
- Helps identify multiple VS Code instances when Alt+Tabbing
